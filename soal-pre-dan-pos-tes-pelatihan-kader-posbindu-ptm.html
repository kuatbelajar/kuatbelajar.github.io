15. Bukan QuizzzDiketahui :Aku = 4²Mau = 5²Off = 3³Soalnya = 5²PTM = 6⁴Maka : Aku + Mau + Off - Soalnya - PTM = ……Nt : kek biasanya ……​


Jawaban:

Diketahui :

Aku = 4²

Mau = 5²

Off = 3³

Soalnya = 5²

PTM = 6⁴

Aku + Mau + Off - Soalnya - PTM =

4² + 5² + 3³ - 5² - 6⁴

=(4x4) + (5x5) + (3x3x3) - (5x5) - (6x6x6x6)

=16 + 25 +27 - 25 - 1.296

= 41 + 27 - 25 - 1.296

=68 - 25 - 1.296

=43 - 1.296

= -1.253
