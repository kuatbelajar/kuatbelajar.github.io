Penjelasan

Degree of Comparison digunakan untuk menyatakan tingkat perbandingan dari suatu objek/benda atau manusia.

Degree of Comparison terbagi atas tiga, yaitu :

1. Positive

=> membandingkan dua objek di tingkat yang sama

2. Comparative

=> membandingkan dua objek di tingkat yang berbeda

3. Superlative

=> membandingkan satu objek dari keseluruhan

Rumus Degree of Comparison :

1. Positive

- S1 + to be + as + Adjective + as + S2

- S1 + Verb + as + Adverb + as + S2 + Verb

2. Comparative

- S1 + to be + Adjective + -er + than + S2

- S1 + to be + more + Adjective + than + S2

- S1 + Verb + Adverb + -er + than + S2 + Verb

- S1 + Verb + more + Adverb + than + S2 + Verb

3. Superlative

- Subject + to be + the + Adjective + -est + Object

- Subject + to be + the + most + Adjective + Object

- Subject + Verb + the + Adverb + -est + Object

- Subject + Verb + the + most + Adverb + Object

20. sebutkan cities cities fantasi​


Jawaban:

Cerita fantasi sendiri merupakan jenis narasi dimana pengarang bisa mengembangkan kreativitas yang dimiliki karena memang cerita ini hanyalah fantasi dari dirinya sendiri. Secara umum, teks cerita ini hampir sama dengan teks narasi secara keseluruhan, namun memiliki perbedaan pada alur, unsur, serta struktur cerita yang digunakan, dimana cerita biasanya cenderung dilebih-lebihkan.

Cerita fantasi bisa dibilang sebagai sebuah kisah atau cerita karangan dengan alur yang normal, namun sifatnya imajinatif. Sering kali berbagai unsur di dalam cerita tersebut, seperti peristiwa yang terjadi di dalamnya, tidak akan pernah benar-benar terjadi di dunia nyata.

36. A painter painted 222 houses in 2 years. The number of houses painted in the first year is 5 times of that of the second year. How many houses did the painter paint in the second year? houses


[tex]5x + x = 222[/tex]

[tex]6x = 222[/tex]

[tex] \frac{6x}{6} = \frac{222}{6} [/tex]

[tex]x = 37[/tex]

So, the number of houses the painter painted in the second year is 37 houses.
