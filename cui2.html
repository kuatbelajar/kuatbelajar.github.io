Pembahasan

[tex]\boxed{\boxed{\bold{Tata~Nama~Senyawa}}}[/tex]

Untuk mengetahui nama suatu senyawa terlebih dahulu kita harus memahami tentang lambang unsur dan nama unsur kimia. Hal tersebut merupakan ilmu dasar yang harus diketahui untuk memperdalam tentang tata nama senyawa. Penamaan senyawa diatur oleh IUPAC (International Union Pure and Applied Chemistry). berdasarkan kesepakatan oleh para ahli ilmuan didunia.

[tex]\mathbf{Tata~Nama~Senyawa~Anorganik}[/tex]

[tex]\mathbf{1.~Senyawa~Biner~dari~Logam~dan~Non~Logam}[/tex]

a. Logam yang memiliki bilangan oksidasi 1 jenis (Golongan IA, IIA, IIIA)

1) Menyebutkan nama logam nya

2) Menyebutkan nama non logam nya + akhiran –ida

Contoh:

NaCl = natrium klorida

MgF2 = magnesium florida

CaCl2 = kalsium klorida

Na2O = natrium oksida

b. Logam yang memiliki bilangan oksidasi lebih dari 1 jenis (logam transisi)

1) Menyebutkan nama logam nya diikuti angka romawi sesuai dengan bilangan oksidasi unsur logam

2) Menyebutkan nama non logam nya + akhiran –ida  

Contoh :

FeO = besi (II) oksida

Cu2O = tembaga (II) oksida

PbI2 = timbal (II) iodida

[tex]\mathbf{2.~Senyawa~Biner~dari~Non~Logam~dan~Non~Logam}[/tex]

1) Menyebutkan nama Yunani jumlah atom non logam ke-1 diikuti dg nama unsur non logam ke-1

2) Menyebutkan nama Yunani jumlah atom non logam ke-2 diikuti dg nama unsur non logam ke-2 + akhiran –ida

Jumlah Atom dalam Bahasa Yunani

1 Mono

2 Di

3 Tri

4 Tetra

5 Penta

6 Heksa

7 Hepta

8 Okta

9 Nona

10 Deka

NB: untuk awala mono tidak digunakan untuk unsur non logam ke-1

[tex]\mathbf{3.~Senyawa~yang~Mengandung~Ion~Poliatom}[/tex]

a. Logam yang memiliki bilangan oksidasi 1 jenis (Golongan IA, IIA, IIIA)

1) Menyebutkan nama kation logamnya

2) Menyebutkan nama anion poliatom

b. Logam yang memiliki bilangan oksidasi lebih dari 1 jenis (logam transisi)

1) Menyebutkan nama kation logam nya diikuti angka romawi sesuai dengan bilangan oksidasi logam

2) Menyebutkan nama anion poliatom

Beberapa Ion Poliatom

NH4+ Amonium

OH- Hidroksida  

CO32- Karbonat  

NO3- Nitrat  

CH3COOH- Asetat  

PO43- Fosfat

SO42- Sulfat  

[tex]\mathbf{4.~Senyawa~Asam~(H^+)}[/tex]

1) Didahului dengan kata asam (H+ ) / kation

2) Menyebutkan nama anion nya

Contoh :

HCl = asam klorida

CH3COOH = asam asetat

H2SO4 = asam sulfat

HNO3 = asam nitrat

[tex]\mathbf{5.~Senyawa~Basa~(OH^-) }[/tex]

1) Menyebutkan nama kationnya

2) Menyebutkan nama anionnya (hidroksida = OH-)

Contoh :

NaOH = natrium hidroksida

Fe(OH)3 = besi (III) hidroksida

Ca(OH)2 = kalsium hidroksida

1) Menyebutkan nama logam nya diikuti angka romawi sesuai dengan bilangan oksidasi unsur logam

9. Pada reaksi redoks: 2 CuSO4 + 4 KI → 2 K2SO4 + 2 CuI2 + I2. Hasil oksidasinya adalah....


Jawaban:

I2

Penjelasan:

2CuSO4 + 4KI → 2K2SO4 + 2CuI + I2

Hasil oksidasi yaitu senyawa yang berada di sebelah kanan panah dan mengalami kenaikan biloks.

Kiri panah

CuSO4

Cu = +2 (krn merupakan kation bermuatan +2)

O = -2 (krn merupakan anion bermuatan -2)

S = +6 (dihitung seperti di bawah ini)

CuSO4 = 0

1.Cu + 1.S + 4.O = 0

(1.+2) + ( 1.S) + (4.-2) = 0

(+2) + S + (-8) = 0

+2 -8 +S = 0

S - 6 = 0

S = +6

KI

K = +1 (krn merupakan anion bermuatan +1)

I = -1 (krn merupakan anion bermuatan -1)

Kanan panah

K2SO4

K = +1 (krn merupakan kation bermuatan +1)

O = -2 (krn merupakan anion bermuatan -2)

S = +6 (dihitung seperti di bawah ini)

K2SO4 = 0

2.K + 1.S + 4.O = 0

(2.+1) + ( 1.S) + (4.-2) = 0

(+2) + S + (-8) = 0

+2 -8 +S = 0

S - 6 = 0

S = +6

CuI

I = -1 (krn merupakan anion bermuatan -1)

Cu = +1 (dihitung seperti di bawah ini)

CuI = 0

1.Cu + 1.I = 0

1.Cu + 1.-1 = 0

Cu -1 = 0

Cu = +1

I2

I2 = 0 (krn merupakan unsur bebas)

————————————————————

Kiri panah Kanan panah

Cu +2 +1

S +6 +6

O -2 -2

K +1 +1

I -1 -1 dan 0

Yang mengalami oksidasi adalah unsur I pada I2, dari -1 menjadi 0.

Sehingga hasil oksidasi adalah I2.

Tata Nama Senyawa

Untuk mengetahui nama suatu senyawa terlebih dahulu kita harus memahami tentang lambang unsur dan nama unsur kimia. Hal tersebut merupakan ilmu dasar yang harus diketahui untuk memperdalam tentang tata nama senyawa. a) Cu₂O

b) CuI₂

c) SnCl₄

Tata nama senyawa menurut sistem Stock

Terlebih dahulu kita akan menentukan tata nama senyawa untuk senyawa biner yaitu terdiri 2 unsur.

a) Cu₂O

b) CuI₂

c) SnCl₄

Pada ketiga senyawa diatas merupakan logam (Cu, Sn) yang memiliki bilangan oksidasi lebih dari satu. Oleh karena itu, untuk menentukan tata nama nya menggunakan :

Menyebutkan nama logam nya diikuti angka romawi sesuai dengan bilangan oksidasi unsur logam Menyebutkan nama non logam nya + akhiran –ida  

a) Cu₂O

2. biloks Cu + biloks O = 0

2. biloks Cu + (-2) = 0

biloks Cu = +1

===> Nama senyawa = Tembaga (I) oksida

b) CuI₂

biloks Cu + 2. biloks I = 0

biloks Cu + 2. (-1) = 0

biloks Cu = +2

===> Nama senyawa = Tembaga (II) iodida

c) SnCl₄

biloks Sn + 4. biloks Cl = 0

biloks Sn + 4. (-1) = 0

biloks Sn = +4

===> Nama senyawa = Timah (IV) klorida

Kemudian untuk tata nama senyawa poliatom yaitu senyawa NaCIO₃ yang terdapat logam (Na) dengan satu bilangan oksidasi maka tata nama nya menggunakan :

Menyebutkan nama kation logamnya Menyebutkan nama anion poliatomPaling belakang menuliskan angka romawi dari bilangan oksidasi unsur yang ada di tengah

d) NaCIO₃

biloks Na + biloks Cl + 3. biloks O = 0

(+1) + biloks Cl + 3. (-2) = 0

biloks Cl = +5

===> Nama senyawa = Natrium klorat (V)

Berdasarkan penjelasan diatas, kesimpulan yang kita dapatkan adalah

Cu₂O = tembaga (I) oksida

CuI₂ = tembaga (II) iodida

SnCl₄ = timah (IV) klorida

NaClO₃ = natrium klorat (V)
