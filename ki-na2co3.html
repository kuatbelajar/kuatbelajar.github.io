Cu²⁺ + O²⁻ → CuO (Tembaga (II) Oksida)Fe³⁺ + SO₄²⁺ → Fe₂(SO₄)₃ (besi (III) sulfat)

26. larutan na2co3 memiliki ph = 8 kah2co3 = 0 pangkat min 4 berapakah konsentrasi na2co3​


Jawaban:

Untuk menyelesaikan masalah ini, kita akan menggunakan persamaan ionisasi dari natrium karbonat dan asam bikarbonat, yaitu:

Na2CO3 → 2Na+ + CO32-

H2CO3 ⇌ HCO3- + H+

Karena larutan memiliki pH 8, maka konsentrasi ion hidrogen (H+) dapat dihitung sebagai berikut:

pH = -log[H+]

8 = -log[H+]

[H+] = 10^-8

Kita juga dapat menghitung konsentrasi HCO3- menggunakan persamaan ionisasi asam bikarbonat:

Kah2co3 = [HCO3-][H+]/[H2CO3]

0,0001 = [HCO3-][10^-8]/[H2CO3]

Diketahui [H2CO3] = [HCO3-] karena asam bikarbonat merupakan buffer, sehingga persamaan dapat disederhanakan menjadi:

0,0001 = [H+]^2/[H2CO3]

0,0001 = (10^-8)^2/[H2CO3]

[H2CO3] = (10^-8)^2/0,0001 = 10^-12

Selanjutnya, kita dapat menghitung konsentrasi natrium karbonat menggunakan hukum kelarutan (Ksp):

Ksp = [Na+][CO32-] = 4,3 x 10^-11

Diketahui bahwa konsentrasi CO32- sama dengan konsentrasi natrium karbonat, sehingga kita dapat menyatakan konsentrasi Na2CO3 sebagai x:

Ksp = [Na+][CO32-] = x^2

4,3 x 10^-11 = x^2

x = √(4,3 x 10^-11) = 2,07 x 10^-6 M

Jadi, konsentrasi Na2CO3 dalam larutan adalah 2,07 x 10^-6 M.

38. 3. Berapa mol Na2Co3 Yang mengandung 3,01 x 10² molekul Na2Co3.​


Jumlah molekul = n x 6,02 x 10²³

keterangan:

n = mol

6,02 x 10²³ = bilangan avogadro

-------

jumlah molekul = n x 6,02 x 10²³

3,01 x 10² = n x 6,02 x 10²³

n = (3,01 x 10²) / (6,02 x 10²³)

n = 1/2 x 10² x 10⁻²³

n = 0,5 x 10⁻²¹

n = 5 x 10⁻²² mol

Jadi, Na2CO3 yang mengandung 3,01 x 10² molekul memiliki 5 x 10⁻²² mol
