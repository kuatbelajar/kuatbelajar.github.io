5. nyatakan deret 4+16+64+256+1024 dalam bentuk notasi sigma​


Bentuk notasi sigma untuk deret 4 + 16 + 64 + 256 + 1024 adalah

4 + 16 + 64 + 256 + 1024 = [tex]\left \sum {_{i =1} ^{5}} 2^{2i} \right[/tex]

Nilai tersebut diperoleh dari perhitungan besar suku ke- n barisan geometri. Simak pembahasan berikut.

Pembahasan

Besar suku ke- n barisan geometri dirumuskan sebagai berikut:

Un = arⁿ⁻¹

dengan:

a = suku awal

r = rasio

n = banyak suku

Barisan geometri merupakan barisan perkalian dan pembagian.

Sedangkan notasi sigma untuk barisan geometri adalah

U₁ + U₂ +U₃ + .. + Un = [tex]\left \sum {_{i =1} ^{n}} Ui \right[/tex]

dengan n adalah banyak suku.

Dari soal diketahui deret bilangan berikut:

4 + 16 + 64 + 256 + 1024

pola yang terbentuk dari barisan bilangan tersebut adalah

4        16      64      256       1024

   ×4      ×4      ×4         ×4

Dari pola barisan diketaui bahwa deret tersebut merupakan deret geometri, dengan:

a = 4

r = 4

n = 5

Sehingga besar suku ke- n deret tersebut diperoleh sebagai berikut:

Un = arⁿ⁻¹

Un = 4 × 4ⁿ⁻¹

Un = 2² × (2²)ⁿ⁻¹

Un = 2² × 2²ⁿ⁻²

Un = 2²⁺²ⁿ⁻²

Un = 2²ⁿ

Maka notasi sigma untuk deret geometri tersebut adalah

4 + 16 + 64 + 256 + 1024 = [tex]\left \sum {_{i =1} ^{n}} Ui \right[/tex]

4 + 16 + 64 + 256 + 1024 = [tex]\left \sum {_{i =1} ^{5}} 2^{2i} \right[/tex]
