2. Jenis resensi yang membahas secara detail di setiap bagian atau bab sebuah buku disebut resensi


Jenis resensi yang membahas secara detail di setiap bagian atau bab sebuah buku disebut resensi Deskriptif. Biasanya resensi deskriptif juga mengulas atau menilai nilai-nilai yang kurang tepat pada sebuah karya ilmiah.

Pembahasan

Resensi merupakan karya ilmiah yang terdiri dari tulisan ilmiah dan didalamnya mencakup isi sebuah buku dari karya ilmiah yang membahas tentang kelemahan dan kelebihan karya sastra tersebut sebagai informasi kepada pembaca. Dalam resensi pada karya ilmiah atau sebuah buku memiliki beberapa jenis seperti :

Resensi Informatif, merupakan resensi yang menyampaikan maksud atau isi dari resensi secara umum dan secara singkat dari keseluruhan isi buku sebagai bentuk informasi kepada pembaca.Resensi Deskriptif, merupakan resensi yang membahas secara detail pada setiap bagian atau babnya dari sebuah karya ilmiah atau buku.Resensi Kritis, merupakan resensi yang memiliki bentuk seperti ulasan secara detail berdasarkan metodologi ilmu pengetahuan tertentu.

Kesimpulannya, Resensi yang membahas secara rinci pada setiap bagian buku adalah jenis resensi deskriptif.

6. diketahui nilai twk kelas A: 85,95,75,P,90,85,105,85,95,115 jika rata" data tersebut adalah 90 nilai P? ​


Penjelasan:

Statistika

Mean [ Rata Rata ]

X = (x¹ + x² + x³ + .... + xⁿ)/ n

X = mean [rata rata]

x¹ , x² = jumlah data

n = banyaknya frekuensi data

sehingga :

Diketahui nilai TWK Kelas A

85 , 95, 75, P, 90, 85, 105, 85 , 95 , 115.

Jika nilai rata rata tersebut adalah 90, maka nilai

P?

misalkan nilai P = x⁴

↓

X = (x¹+x²+x³+x⁴+x⁵+x⁶+x⁷+x⁸+x⁹+x¹⁰)/n

90 = (85+95+75+P+90+85+105+85+95+115)/10

90 . 10 = 830 + P

900 = 830 + P

P = 900 - 830

P = 70

Jadi, nilai P yang memenuhi adalah P = 70

26. Tuliskan syntax Php untuk menghapus data siswa berdasarkan ID siswa​


Jawaban:

berikut code nya :

<?php

// Koneksi ke database

$conn = mysqli_connect("localhost", "username", "password", "nama_database");

// Menangkap ID siswa yang akan dihapus dari parameter GET

$id_siswa = $_GET["id"];

// Membuat query untuk menghapus data siswa berdasarkan ID

$query = "DELETE FROM siswa WHERE id_siswa = $id_siswa";

// Menjalankan query untuk menghapus data

if (mysqli_query($conn, $query)) {

echo "Data siswa berhasil dihapus";

} else {

echo "Gagal menghapus data siswa: " . mysqli_error($conn);

}

// Menutup koneksi ke database

mysqli_close($conn);

?>

Penjelasannya :

1. Pertama-tama kita melakukan koneksi ke database dengan menggunakan fungsi `mysqli_connect()` . Di sini, kamu harus mengganti "localhost", "username", "password", dan "nama_database" dengan nilai yang sesuai untuk koneksi ke database kita.

2. menangkap ID siswa yang akan dihapus dari parameter GET dengan menggunakan variabel `$_GET["id"]`.

3. Setelah itu, kita membuat query SQL untuk menghapus data siswa berdasarkan ID menggunakan perintah `DELETE FROM`. Di sini, kita menggunakan variabel `$id_siswa` untuk menggantikan nilai ID siswa yang kita tangkap sebelumnya.

4. Kita mengeksekusi query menggunakan fungsi `mysqli_query()`. Jika query berhasil dijalankan, kita akan menampilkan pesan "Data siswa berhasil dihapus". Jika tidak, kita akan menampilkan pesan error menggunakan fungsi `mysqli_error()`.

5. Terakhir, kita menutup koneksi ke database menggunakan fungsi 'mysqli_close()'.

<?php

// koneksi ke database

$servername = "localhost";

$username = "username";

$dbname = "myDB";

if (!$conn) {

 die("Koneksi gagal: " . mysqli_connect_error());

// ambil ID siswa yang akan dihapus dari parameter URL

$id = $_GET['id'];

if (mysqli_query($conn, $sql)) {

 echo "Error: " . $sql . "<br>" . mysqli_error($conn);

// tutup koneksi

Jika tidak, pesan error dicetak ke layar.Koneksi ditutup dengan fungsi mysqli_close().
