11. Dimas mengikuti sebuah tes yang terdiri atas 50 soal. Penilaian tes tersebut sebagai berikut : Untuk jawaban benar diberi nilai 2, jawaban salah diberi nilai -1, dan untuk soal yang tidak dijawab diberi nilai 0. Dimas menjawab 45 soal. Jika 34 soal dijawab dengan benar, 11 soal dijawab salah, berapa nilai yang diperoleh Dimas?


Diketahui

- ada 50 soal

- benar nilainya 2 (34 soal)

- salah nilainya -1 ( 11 soal)

- tidak di jawab nilainya 0 ( 5 soal )

Ditanya

Berapa nilai yang di peroleh Dimas?

Jawab

= ( 2 × 34 ) + ( -1 × 11 ) + ( 5 x 0 )

= 68 + (-11) + 0

= 68 - 11 + 0

= 57

Jadi, nilai yang di peroleh Dimas adalah 57

Jawaban:

nilai dimas adalah = 68

Penjelasan dengan langkah-langkah:

diketahui:

jumlah soal : 50 soal

soal dengan jawaban benar : 2 point

soal dengan jawaban salah : -1 point

soal yang tidak di jawab : 0 point

banyak soal yang di jawab oleh dimas : 45 soal

banyak soal yang di jawab benar oleh dimas : 34 soal

banyak soal yang di jawab salah oleh dimas : 11 soal

Ditanyakan :

berapa nilai yang di peroleh dimas?

penyelesain :

50 - 45 = 5

45 - 11 = 34

= 34 x 2 point

= 68 point

45 - 34 = 11

= 11 x - 1

= - 11

jadi 5 soal point = 0

11 soal salah point = -11

34 soal benar point = 68

jawaban = nilai dimas adalah 68 point
