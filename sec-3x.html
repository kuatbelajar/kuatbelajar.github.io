1. Sec 3x = sec 30° 0°≤x≤ 360°


Persamaan Trigonometri

Pembahasan

[tex] \sec3x = \sec30 \\ \frac{1}{ \cos3x} = \frac{2}{3} \sqrt{3} \\ \cos3x = \frac{1}{2} \sqrt{3} \\ 3x = 30[/tex]

cos x = cos a => x1.2 = ± a + k.360°

• untuk k = 0

3x = 30° + (0)360°

x = 10°

• untuk k = 1

3x = 30° + (1)360°

x = 130°

atau

3x = -30° + (1)360°

x = 110°

• untuk k = 2

3x = 30° + (2)360°

x = 250°

atau

3x = -30° + (2)360°

x = 230°

Himpunan penyelesaian : {10°, 110°, 130°, 230°, 250°}
