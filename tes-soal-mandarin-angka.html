7. 412 ditulis Dengan angka Mandarin​


Jawab:

四百十二 (Sìbǎi shí èr)

Penjelasan:

Bahasa Mandarin angka 1-10:

Satu (1) → 一 (Yī)Dua (2) → 二 (Èr)Tiga (3) → 三 (Sān)Empat (4) → 四 (Sì)Lima (5) → 五 ( Wǔ)Enam (6) → 六 (Liù)Tujuh (7) → 七 (Qī)Delapan (8) → 八 (Bā)Sembilan (9) → 九 (JiǔSepuluh (10) → 十 (Shí)

·

Untuk menyatakan belasan:

十 (Shí → 10) + Angka (untuk satuan)

Untuk menyatakan puluhan:

Angka (untuk puluhan) + 十 (Shí → 10) + Angka (untuk satuan)

Untuk menyatakan ratusan:

Angka (untuk ratusan) + 百 (Bǎi → 100) + Angka (untuk puluhan) + Angka (untuk satuan)

·

Sehingga untuk angka 412

→ 4 + 百 + 十 + 2

→ 四 + 百 + 十 + 二

→ 四百十二

→ Sìbǎi shí èr

19. bahasa mandarin dari angka 3


Bahasa Mandarin dari angka 3 adalah 三 (pinyin: sān).

PEMBAHASAN

Angka dalam bahasa Mandarin disebut "数字" (pinyin: shùzì). Berikut adalah beberapa angka di bahasa Mandarin.

⋄ Nol (0)

Hanzi: O / 零Pinyin: líng

⋄ Satu (1)

Hanzi: 一 Pinyin: yī

⋄ Dua (2)

Hanzi: 二 / 两Pinyin: ér / liǎng

⋄ Tiga (3)

Hanzi: 三Pinyin: sān

⋄ Empat (4)

Hanzi: 四Pinyin: sì

⋄ Lima (5)

Hanzi: 五Pinyin: wǔ

⋄ Enam (6)

Hanzi: 六Pinyin: liù

⋄ Tujuh (7)

Hanzi: 七 Pinyin: qī

⋄ Delapan (8)

Hanzi: 八 Pinyin: bā

⋄ Sembilan (9)

Hanzi: 九 Pinyin: jiǔ

⋄ Sepuluh (10)

Hanzi: 十 Pinyin: shí

⋄ Sebelas (11)

Hanzi: 十一 Pinyin: shíyī

⋄ Dua belas (12)

Hanzi: 十二 Pinyin: shí ér

⋄ Dua puluh (20)

Hanzi: 二十Pinyin: ér shí

⋄ Dua puluh satu (21)

Hanzi: 二十一Pinyin: ér shí yī

⋄ Seratus (100)

Hanzi: 一百 Pinyin: yībǎi

Contoh kalimat:

⊱ 我有三本书 // Wǒ yǒusān běn shū.

Saya mempunyai tiga buah buku.

⊱ 你带了三个苹果吗? // Nǐ dàile sān gè píngguǒ ma?

Apakah kamu bawa tiga buah apel?

KESIMPULAN

Angka tiga (3) dalam bahasa Mandarin adalah 三 (pinyin: sān).

22. Apakah bahasa mandarinnya angka satu?


Bahasa Mandarin dari angka satu adalah 一 (yī).

Kosakata Mengenai Angka.

1 = 一 dibaca sebagai (yī).

2 = 二 dibaca sebagai (èr).

3 = 三 dibaca sebagai (sān).

4 = 四 dibaca sebagai (sì).

5 = 五 dibaca sebagai (wǔ).

6 = 六 dibaca sebagai (liù).

7 = 七 dibaca sebagai (qī).

8 = 八 dibaca sebagai (bā).

9 = 九 dibaca sebagai (jiǔ).

10 = 十 dibaca sebagai (shí).

Pembahasan

Angka-angka dalam bahasa Mandarin ditulis menggunakan sistem penulisan aksara Hanzi. Namun Angka-angka yang ditulis dalam bahasa Mandarin juga dapat ditulis dalam angka Jepang. Angka Jepang yang ditulis menggunakan Aksara Hanzi itu dinamakan Kanji. Sudah saya jelaskan bahwa Aksara Kanji yang masuk ke Jepang disebut sebagai Kanji. Namun tidak semua Aksara Hanzi dapat masuk ke Jepang, karena bertahun-tahun dulu sudah dibatasi masuknya Aksara Hanzi ke Negara Jepang.

31. Buatlah Percakapan tentang angka memakai bahasa mandarin!


Jam itu termasuk Angka. Jadi saya membuat percakapan mengenai jam.

Contoh percakapannya :

A : 下午好，现在几点了？

B : 下午好，现在是下午三点三十分了。

A : 我的天！我的表迟到了三十分钟。

B : 在这里，也许我可以纠正先生。

A : 好的，谢谢。

B : 别客气。

Terjemahan :

A : Selamat sore, sekarang jam berapa?

B : Selamat sore, sekarang jam 15.30

A : Astaga! Jam tanganku telat 30 menit.

B : Sini, mungkin saya bisa betulkan, Pak.

A : Baiklah, Terima kasih.

B : Sama-sama.

Pembahasan

Waktu dalam bahasa Mandarin :

早晨 (zǎochén) : Pagi.下午 (xiàwǔ) : Siang.中午 (zhōngwǔ) : Senja/Malam.晚上 (wǎnshàng) : Malam.

Angka dalam bahasa Mandarin :

一 (yī) : Satu.二 , 两 (èr, liǎng) : Dua.三 (sān) : Tiga.四 (sì) : Empat.五 (wǔ) : Lima.六 (liù) : Enam.七 (qī) : Tujuh.八 (bā) : Delapan.九 (jiǔ) : Sembilan.十 (shí) : Sepuluh.

Cara menyatakan jam :

Waktu + Angka (Jam) + 点

Contoh :

下午两点 (Xiàwǔ liǎng diǎn) artinya Jam dua siang.

Kosakata :

- 下午 (Xiàwǔ) : siang.

- 两 (liǎng) : Angka 2.

- 点 (diǎn) : Jam.
