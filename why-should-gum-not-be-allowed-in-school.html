1. What is purpose of the text? (Apa tujuan dari teks tersebut )

To convince the readers that smoking is very dangerous for human's health

2. What kind of the text ? ( Apa jenis teks tersebut?)

The text is analytical exposition

3. What are the generic structure of the text? ( Apa susunan umum dari teks tersebut?)

The generic structures of analytical exposition text are thesis, arguments and reiteration ( conclusion )

DATA ANALYSIS :

Smoking in restaurants is just not on. It must not be allowed because it is rude, harmful to others and dangerous for the smokers.  ( THESIS , THE WRITER PLOTS THE MAIN ISSUE  TO BE IN THE CASE STUDY )

Firstly, smoking in a restaurant is impolite. The smell of the smoke affects all people and can turn them off their food. People pay to taste good food and not to be put off by foul smelling smoke. Another reason smoking should not be allowed in restaurant is the harm it can do to others. Passive smoking that is breathing in smoke made by a smoker can lead to asthma attacks and even cancer.Finally, smoking is dangerous and a health risk to the smokers. Cigarettes cause heart and lung disease and people should not smoke anywhere, not just in restaurants.  ( THE WRITER'S ARGUMENT  & THE REALISTIC EVIDENCE HAS BEEN SHOWN )

Therefore, smoking in restaurants is impolite, harmful to others and a health risk to the smokers and should not be allowed in any restaurants.( REITERATION , AS THE WRITER'S  PURPOSE EMPHASIZING )

Penjelasan:

ANALYTICAL EXPOSITION TEXT

~ Genre teks ini menjabarkan pandangan penulis terhadap suatu masalah yang dianggap penting untuk didiskusikan.

The social function / communicative purpose  

~To convince the readers that there is an important thing in the case.

( untuk meyakinkan para pembaca bahwa ada hal penting dalam suatu kasus  )

Generic structure  

1. Thesis  ( general statement )

Paragraf pertama ini menjabarkan rumusan masalah yang didiskusikan berdasarkan cara pandang si penulis .

2. Arguments

Paragraf selanjutnya menjabarkan opini-opini si penulis  beserta didukung informasi  realistis ,  pembuktian fakta guna mendukung argumentasinya.

3. Reiteration

Paragraf akhir berisi kesimpulan / pengulasan kembali  / penekanan dari tujuan si penulis yang telah dipaparkan pada thesis nya.

Language features  ( Unsur kebahasaan )

~ Using internal conjunction to elaborate the argument ( firstly, secondly, ....... )

~  Using temporal conjunction  ( relative clause ) (  on the other hand , meanwhile.......)

~ Using -If conditional sentence

~ Using contrastive conjunction ( but, however .....)

~ Using causal conjunction ( because.........., caused by ....)

~ Using simple present tense  

Karena berdasarkan fakta kebenaran, tenses keseluruhan menggunakan simple present tense.

~ Using passive voice ( tindakan aksi difokuskan utama )

~ Using mental  expression words  ( lebih ke opini si penulis )

~ Using the abstract noun ( we,   they,  passive smokers........)

~ Using modal verbs     ( should........can........)
