33. Sita mengikuti sebuah tes. Dalam tes tersebut, Sita harus mengerjakan 40 soal. Setelah waktu tersisa 15 menit ada 4 soal yang belum dikerjakan. Banyaknya soal yang telah dikerjakan Sita ada … %.PLIIS KK JAWAB!!!!​


Jawab:

90%

Penjelasan dengan langkah-langkah:

Diketahui:

Jumlah soal ada 40 soal.

Sisa soal yang belum dikerjakan 4 soal.

Ditanyakan:

Berapa persen soal yang sudah dikerjakan Sita?

Penjelasan:

Artinya, Sita sudah mengerjakan 36 soal dari total 40 soal, hitungannya:

Jumlah total soal - sisa soal yang belum dikerjakan = 40 - 4 = 36

Untuk mengetahui berapa persennya, caranya:

Jumlah soal yang sudah Sita kerjakan / Total jumlah soal x 100% =

36/40 x 100% =

3600/40 = 90%

Jadi jumlah soal yang sudah dikerjakan oleh Sita mencapai 90%

x - 2 = 0

Untuk akar-akar kedua

x + 10 = 0

x = 0 - 10

x = - 10

Sehingga diperoleh:

HP = { 2, - 10 }

Nomor 2

x² + x - 6 = 0

x(x + 3) - 2(x + 3)

(x - 2)(x + 3) = 0

Untuk akar-akar pertama

x - 2 = 0

x = 0 + 2

x = 2

Untuk akar-akar kedua

x + 3 = 0

x = 0 - 3

x = - 3

Sehingga diperoleh:

HP = { x₁, x₂ }

HP = { 2, - 3 }

Penjelasan dengan langkah-langkah:

Bentuk umum persamaan kuadrat ax² + bx + c = 0

Dengan a ≠ 0

Mencari akar akar persamaan kuadrat dari :

1). x² + 8x - 20 = 0 → menggunakan cara pemfaktoran

x² + 10x - 2x - 20 = 0

x(x + 10) - 2(x + 10) = 0

(x - 2)(x + 10) = 0

x¹ = 2 dan x² = –10

Jadi, Himpunan penyelesaian nya adalah

HP = {( 2 dan –10 )}

....
