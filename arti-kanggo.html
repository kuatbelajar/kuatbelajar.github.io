23. kanggo nyekel lintang ono langit tolong artikan dalam bahasa jawa


Kanggo nyekel lintang ono langit artinya untuk memegang bintang yang ada di langit.

Pembahasan

Arti per kata :

Kanggo = untuk

nyekel = memegang

lintang = bintang

ono = ada

langit = langit

Kesimpulan :

Kanggo nyekel lintang ono langit artinya untuk memegang bintang yang ada di langit.

40. BAHASA JAWA : 1. Kuping gunane kanggo ... 2. Mripat gunane kanggo ... 3. Tangan gunane kanggo ... 4. Sekel gunane kanggo .... 5. Cangkem gunane kanggo ....

2. Mripat gunane kanggo Ndeleng utawa nyawang

3. Tangan gunane kanggo nyekel

4. Sikel gunane kanggo mlaku

5. Cangkem gunane kanggo ngomong lan dhahar/maem

Pembahasan :

Perangane awak yaiku bagian bagiane awak atau dalam bahasa Indonesia bisa diartikan sebagai bagian bagian tubuh

Perangane awak antara lain ing ngisor iki :

Mripat ( Mata ) gunane kanggo ndeleng ( melihat )Rambut gunane kanggo nutupi sirah ( menutupi kepala )Irung ( hidung ) gunane kanggo ambegan ( bernafas )Cangkem ( mulut ) gunane kanggo dhahar lan ngomong ( makan dan berbicara )Untu ( gigi ) gunane kanggo nyakot, mamah ( menggigit, mengunyah )Kuping ( telinga ) gunane kanggo ngrungokake ( mendengarkan )Tangan gunane kanggo nyekel ( memegang )Pundak gunane kanggo manggol/mikul ( memikul )Sikil ( Kaki ) gunane kanggo mlaku ( berjalan )
