Buatlah program untuk menghitung gaji pegawai Input : nama, golongan Proses : · Jika golongan A maka gaji pokok =Rp 1.500.000, tunjangan =5% * gaji pokok .jika gol.B maka gaji pokok = Rp.1.700.000,tunjangan = 10%* gaji pokok .jika gol.C maka gaji pokok = Rp.2.000.000, tunjangan = 15%*gaji pokok Output : Nama,gaji pokok,tunjangan serta gaji bersih Petunjuk : gunakan array dan petualangan


program untuk menghitung gaji pegawai, program ditulis dalam bahasa C

#include <stdio.h>

int main()

{

    char nama[50];

    char golongan;

    int gaji[3] = { 1500000, 1700000, 2000000 };

    int persentunjangan[3] = { 5, 10, 15 };

    int gajibersih, tunjangan;

    printf("Masukan nama Anda : ");

    scanf("%s", nama);

    printf("Masukan golongan (A, B, atau C) : ");

    scanf("%c", &golongan);

    golongan = golongan - 'A';

    tunjangan = gaji[ golongan ] * persentunjangan[ golongan ] / 100;

    gajibersih = gaji[ golongan ] + tunjangan;

    printf("Nama : %s\n", nama);

    printf("Gaji pokok : %i\n", gaji[ golongan ]);

    printf("Tunjangan : %i\n", tunjangan);

    printf("Gaji bersih : %i\n", gajibersih);

    return 0;

}

Pembahasan

Ada bagian yang aneh pada soal, "gunakan array dan petualangan". Array adalah bentuk variabel dengan alokasi memory untuk banyak nilai. Yang saya bingung adalah istilah "petualangan" disini. Setahu saya tidak ada istilah "petualangan" dalam bahasa pemrograman, mohon dikoreksi jika memang terjadi typo.

Pada solusi pemrograman yang dibuat, kita memanfaatkan golongan untuk menjadi index dari Array.

Pada potongan code berikut golongan = golongan - 'A';

30. Hotel MEGA ASRI memberi gaji pada karyawannya setiap bulan dengan ketentuan setiap karyawan selain mendapat gaji pokok juga mendapat tunjangan keluarga sebesar 1\5 gaji pokok ditambah Rp50.000,00 dan tunjangan kesehatan sebesar 1/2 dari tunjangan keluarga. Arinda seorang pegawai hotel MEGA ASRI memiliki gaji pokok sebesar Rp2.000.000,00. Besar gaji dan tunjangan yang diterima Arinda setiap bulan adalah ….


Jawab:

Arinda akan menerima gaji dan tunjangan sebesar Rp2.675.000,00 setiap bulan.

Penjelasan dengan langkah-langkah:

Untuk menghitung besar gaji dan tunjangan yang diterima Arinda setiap bulan, berikut adalah langkah-langkahnya:

Hitung tunjangan keluarga:

Tunjangan keluarga = 1/5 x gaji pokok + Rp50.000,00

Tunjangan keluarga = 1/5 x Rp2.000.000,00 + Rp50.000,00

Tunjangan keluarga = Rp400.000,00 + Rp50.000,00

Tunjangan keluarga = Rp450.000,00

Hitung tunjangan kesehatan:

Tunjangan kesehatan = 1/2 x tunjangan keluarga

Tunjangan kesehatan = 1/2 x Rp450.000,00

Tunjangan kesehatan = Rp225.000,00

Hitung total gaji dan tunjangan:

Total gaji dan tunjangan = gaji pokok + tunjangan keluarga + tunjangan kesehatan

Total gaji dan tunjangan = Rp2.000.000,00 + Rp450.000,00 + Rp225.000,00

Total gaji dan tunjangan = Rp2.675.000,00

Jadi, Arinda akan menerima gaji dan tunjangan sebesar Rp2.675.000,00 setiap bulan.
