10. 2. A fashion magazine was readed by she inthe lobby​


A fashion magazine was read by her in the lobby

= Sebuah majalah mode dibaca oleh dia di lobi.

# Ini adalah kalimat pasif the simple past tense

# Rumus kalimat pasif the simple past tense =

( + ) SUBJECT - WAS / WERE - VERB 3

* Subject = A fashion magazine

* To Be = was

* Verb 3 = Read

# Membaca

* Verb 1 = Read

* Verb 2 = Read

* Verb 3 = Read

**********************

# Oleh dia = By her ( Bukan By She; Karena DIA di sini berfungsi sebagai Obyek.

# Dia perempuan ( sebagai Subyek ) = SHE

# Dia perempuan ( sebagai Obyek ) = HER.

# SHE was my friend ( DIA adalah teman saya )

# I want to phone HER ( Saya mau menelepon DIA )

12. Active : My father reads a daily newspaper. Passive : …. a. A daily newspaper is readed by my father. b. A daily newspaper are read by my father. c. A daily newspaper are readed by my father. d. A daily newspaper is read by my father. e. A daily newspaper was read by my father. ​


Jawaban:

daily newspaper is read by My Father

Penjelasan:

Berdasarkan Rumus Simple Present

(+) S + VERB 1 (-s/es) + O

AV: My Father reads a daily newspaper

= Ayahku membaca Sebuah koran harian

(+) S + To Be (is/am/are) + VERB 3 + by

Pav: A daily newspaper + is + read + by My Father

= Sebuah koran harian dibaca oleh Ayahku
