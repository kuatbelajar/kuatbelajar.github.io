4. ∫sin2x cos2x dx dan ∫sin 5x cos2x dx tolong pake penjelasan!!


SOAL 1

Misalkan [tex]\sin 2x = u[/tex]

turunkan kedua ruas

[tex]\sin 2x = u[/tex]

[tex]2 \cos 2x ~dx = 1~ du[/tex]

[tex]dx = \frac{du}{2\cos 2x}[/tex]

Sehingga diperoleh

[tex]\int\limits {\sin2x.\cos2x} \ dx[/tex]

[tex]= \int\limits {u.\cos2x} \ \frac{du}{2\cos 2x}[/tex]

[tex]= \int\limits {\frac{1}{2}u } \ du[/tex]

[tex]= \frac{1}{4} u^2 + c[/tex]

[tex]= \boxed{\frac{1}{4} \sin^2 2x + c}[/tex]

.

SOAL 2

Ubah bentuk soal, gunakan rumus dibawah

[tex]\boxed{\boxed{\sin a . \cos b = \frac{1}{2} (\sin(a + b) - \sin(a - b))}}[/tex]

Sehingga

[tex]\int\limits {\sin 5x.\cos 2x} \ dx[/tex]

[tex]= \int\limits { \frac{1}{2} (\sin(5x + 2x) - \sin(5x - 2x))} \ dx[/tex]

[tex]= \int\limits { \frac{1}{2} (\sin 7x - \sin 3x)} \ dx[/tex]

[tex]=\frac{1}{2} \int\limits { \sin 7x - \sin 3x} \ dx[/tex]

[tex]=\frac{1}{2} \left(-\frac{1}{7} \cos7x +\frac{1}{3} \cos 3x \right) + c[/tex]

[tex]= \boxed{-\frac{1}{14} \cos7x +\frac{1}{6} \cos 3x + c}[/tex]

5. Integral dari 1-cos2x / sin2x


[tex]\displaystyle \int \frac{1-\cos{2x}}{\sin{2x}} \ dx[/tex]

Pecah integral jadi 2 bagian,
[tex]\displaystyle \int\frac{1}{\sin{2x}} \ dx - \int \frac{\cos{2x}}{\sin{2x}} \ dx[/tex]

Untuk integral kedua, gunakan metode substitusi:
[tex]u := \sin{2x} \implies \frac{1}{2} \ du = \cos{2x} \ dx[/tex]

[tex]\displaystyle \int \frac{1}{\sin{2x}} \ dx - \frac{1}{2}\int \frac{1}{u} \ du[/tex]
[tex]\displaystyle \int \frac{1}{\sin{2x}} \ dx - \frac{1}{2}\ln{u}[/tex]

Ubah kembali u = sin(2x):
[tex]\displaystyle \int \frac{1}{\sin{2x}} \ dx - \frac{1}{2}\ln{\sin{2x}}[/tex]

Untuk integral pertama, ubah menjadi csc(2x):
[tex]\displaystyle \int \csc{2x} \ dx - \frac{1}{2}\ln{|\sin{2x}|}[/tex]

Integral csc adalah: [tex]\ln|\csc{x}+\cot{x}| + C[/tex], saya akan jelaskan bagaimana caranya:
Kalikan integral csc dengan [tex]\frac{\csc{2x}+\cot{2x}}{\csc{2x}+\cot{2x}}[/tex] sehingga:

[tex]\displaystyle \int \csc{2x} \frac{\cot{2x}+\csc{2x}}{\cot{2x}+\csc{2x}} \ dx - \ln|\sin{2x}|[/tex]

[tex]\displaystyle \int \frac{\csc{2x}\cot{2x}+\csc^2{2x}}{\cot{2x}+\csc{2x}} \ dx - \ln|\sin{2x}|[/tex]

Gunakan metode substitusi:
[tex]v := \cot{2x}+\csc{2x}\implies \frac{1}{2}\ dv = \csc{2x}\cot{2x}+\csc^2{x} \ dx[/tex]
sehingga integral tadi bisa ditulis:
[tex]\displaystyle \frac{1}{2}\int \frac{1}{v} \ dv - \ln|\sin{2x}|[/tex]

Ambil integral 1/v = ln|v|
[tex]\displaystyle \frac{1}{2} \ln|v| - \frac{1}{2}\ln|\sin{2x}|[/tex]

Kembalikan lagi nilai v = cot(2x)+csc(2x):
[tex]\displaystyle \frac{1}{2}\ln|\cot{2x}+\csc{2x}| - \frac{1}{2}\ln|\sin{2x}|[/tex]

Sederhanakan dengan aturan logaritma:
[tex]\displaystyle = \frac{1}{2}(\ln|\cot{2x}+\csc{2x}|-\ln|\sin{2x}|) = \frac{1}{2}\ln\left|\frac{\cot{2x}+\csc{2x}}{\sin{2x}}\right|[/tex]

Tambahkan konstan pada hasil akhir:
[tex]\displaystyle = \boxed{\frac{1}{2}\ln\left|\frac{\cot{2x}+\csc{2x}}{\sin{2x}}\right| + C}[/tex]

atau
[tex]\displaystyle = \boxed{\ln\left|\sqrt{\frac{\cot{2x}+\csc{2x}}{\sin{2x}}}\right| + C}[/tex]

26. berapa bentuk sederhana dari sin2x + cos2x = ?


Bentuk sederhananya adalah 1

Pembahasan

Trigonometri adalah materi yang membahas bentuk panjang sisi atau besar sudut yang dimana didasarkan pada bangun segitiga. Dalam trigonometri tersebut terdapat aturan sinus, cosinus dan tangen yang digunakan menentukan bagian-bagian dari segitiga tersebut. Aturan tersebut juga mempunyai identitas trigonometri yang menjadikan relasi pada fungsi yang dimiliki untuk melakukan pergantian variabel dan konstanta.

Rumus Identitas Trigonometri

[tex]\boxed{\boxed{1. \: \sin^{2} \: x + cos {}^{2} \: x = 1 }} \\ \boxed{\boxed{2. \: sin {}^{2} \: x = 1 - {cos}^{2} \: x}} \\ \boxed{\boxed{3. \:cos^{2} \: x = 1 - {sin}^{2} \: x }} \\ \boxed{\boxed{4. \: tan \: x = \frac{sin \: x}{cos \: x} }} \\ \boxed{\boxed{5. \: cot \: x = \frac{cos \: x}{sin \: x} }} \\ \boxed{\boxed{6. \: sec \: x = \frac{1}{cos \: x} }} \\ \boxed{\boxed{7. \: csc \: x = \frac{1}{sin \: x} }} \\ \boxed{\boxed{ 8. \: {sec}^{2} = {tan}^{2} x + 1}} \\ \boxed{\boxed{9. \: csc {}^{2} \: x = {cot}^{2} \: x + 1}}[/tex]

Penyelesaian Soal

Diketahui :

Bentuk trigonometri = [tex]sin^{2} \:x + cos^2\:x[/tex]

Ditanyakan :

Bentuk sederhana?

Jawab :

[tex]sin^{2} \: \alpha + cos^2 \: \alpha = 1\\\\sin^2 \: x + cos^2 \: x = 1[/tex]

Jadi, Bentuk sederhananya adalah 1
