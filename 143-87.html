19. Dari 143 siswa,95 siswa senang matematika dan fisika 87 dan 60 siswa senang keduanya.Banyak siswa yang tidak senang matematika maupun fisika adalah berapa.


Jawaban :

21 orang

Penjelasan dengan langkah-langkah :

Diketahui :

Jumlah siswa = 143 orang

Senang matematika = 95 orang

Senang fisika = 87 orang

Senang keduanya = 60 orang

Ditanya :

Banyak siswa yang tidak senang matematika dan fisika

Pembahasan :

Misalkan banyak siswa yang tidak senang matematika dan fisika = x, maka bentuk kalimat matematikanya adalah :

143 = (95 - 60) + (87 - 60) + 60 + x

Selesaikan persamaan x

143 = 35 + 27 + 60 + x

143 = 122 + x

x = 143 - 122

x = 21

Pembuktian :

143 = (95 - 60) + (87 - 60) + 60 + 21

143 = 35 + 27 + 60 + 21

143 = 143

Terbukti.
