15. Pada suatu fungsi, daerah yang semua anggotanya selalu berpasangan adalah... a. domainb. kodomainc. domain dan kodomaind. domain dan rangetolong jawab yang bener, butuh banget​


Pada suatu fungsi, daerah yang semua anggotanya selalu berpasangan adalah C. Domain dan Kodomain.

Pembahasan

Domain adalah himpunan semua input atau argumen yang dapat diterima oleh fungsi, sedangkan kodomain adalah himpunan semua nilai yang mungkin untuk keluaran fungsi. Jadi, jika setiap elemen dalam domain memiliki pasangan yang sesuai dalam kodomain, maka semua anggota fungsi berpasangan dalam daerah yang mencakup domain dan kodomain.

20. diketahui sebuah fungsi F:x -2x+4 domain =domain (-2,-1,0,1,2,3) tentukan daerah hasil range​


Diketahui fungsi f: x ---> -2x + 4 dengan domain {-2, -1, 0, 1, 2, 3}, maka daerah hasil untuk fungsi tersebut adalah {-2, 0, 2, 4, 6, 12}.

Pembahasan

Suatu fungsi dapat dinyatakan dalam bentuk f : x --> f(x). Nilai fungsi untuk setiap nilai x yang diberikan dapat dihitung dengan cara mensubstitusikan nilai x ke dalam rumus fungsi tersebut.

Domain (daerah asal) fungsi adalah himpunan nilai-nilai agar suatu fungsi terdefinisi.

 

Daerah hasil fungsi (range) adalah himpunan nilai-nilai fungsi dari setiap anggota daerah asal (domain).

Penyelesaian

diket:

f: x ---> -2x + 4

domain = {-2, -1, 0, 1, 2, 3}

ditanya:

daerah hasil / range....?

jawab:

    Untuk menentukan daerah hasil suatu fungsi yang diketahui daerah asalnya adalah dengan memasukkan nilai x ke dalam fungsi f(x)

f(x) = -2x + 4

f(-2) = -2(-4) + 4 = 8 + 4 = 12

f(-1) = -2(-1) + 4 = 2 + 4 = 6

f(0) = -2(0) + 4 = 0 + 4 = 4

f(1) = -2(1) + 4 = -2 + 4 = 2

f(2) = -2(2) + 4 = -4 + 4 = 0

f(3) = -2(3) + 4 = -6 + 4 = -2

tulis daerah hasil dari kecil ke besar  

daerah hasil = {-2, 0, 2, 4, 6, 12}

Kesimpulan

Jadi, daerah hasil untuk fungsi tersebut adalah {-2, 0, 2, 4, 6, 12}.

Penjelasan dengan langkah-langkah:

A = { 3, 4 }

B = { 3, 4, 5, 6 }

Relasi = kurang dari

maka, himpunan pasangan berurutannya:

{ ( 3, 4 ), ( 3, 5 ), ( 3, 6 ), ( 4, 5 ), ( 4, 6 ) }
