6. Tolong bantu. Lg try out


Mapel : Fisika
Bab : Gaya

EF = F3 - F2 - F1
EF = 1.500 - 600 - 600 N
EF = 1.500 - 1.200 N
EF = 300 N

W = 18.000 Joule

w = F × s
s = w/F

s = 18.000/300
s = 180/3
s = 60 meter

(D )Mapel : IPA Fisika
Kelas : 8 SMP/VIII SMP
Bab : Gaya dan Usaha 

Pembahasan Soal :
Jika usaha bersama gaya gaya tersebut terhadap benda 18.000 J, Benda berpindah sejauh..

Pembahasan Jawaban :
DiKetahui :
F1 = 600 N ( Kearah Kiri )
F2 = 600 N ( Kearah Kiri )
F3 = 1.500 N ( Kearah Kanan )
w = 18.000 J 

DiTanya :
s =...?

DiJawab :
Sebelum mencari s. Kita harus mencari ∑F
Rumus mencari ∑F
Yaitu : 
∑F = F3 - ( F1 + F2 )
∑F = 1.500 - ( 600 + 600 )
∑F = 1.500 - 1.200
∑F = 300 N ( Kearah Kanan )

Rumus mencari s 
Yaitu :
w = ∑F × s
18.000 = 300 × s
s = 18.000 : 300
s = 60 Meter.
Jadi, benda berpindah sejauh 60 meter ( D )

Semoga Membantu

  Macam bioteknologi
 1.Bioteknologi konvensional
    Bioteknologi yang memanfaatkan mikroorganisme secara langsung untuk     memproduksi makanan.
  Misalnya:Pembuatan kecap dengan bantuan aspergillus wentii
 2.Bioteknologi modern
   Bioteknologi yang memanfaatkan mikroorganisme yang direkayasa, sehingga menjadi    lebih efektif dan efisien

  ~ alel S akan selalu terekspresikan (nampak secara visual dari luar)
2.Hukum Mendel II (Hukum berpasangan secara bebas/independent assortment)
  Bila dua individu mempunyai 2 pasang atau lebih sifat, maka diturunkannya sepasang sifat secara bebas , tidak bergantung pada pasangan sifat yang lain.Dengan kata lain, alel dengan gen sifat yang berbeda tidak saling memengaruhi.
