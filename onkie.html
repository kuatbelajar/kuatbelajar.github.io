Js = Jp : S

Js = jarak sebenarnya

Jp = jarak pada peta

S = skala

1. Js = Jp : S

  Js = 12 : [tex]\frac{1}{250.000}[/tex]

  Js = 12 x [tex]\frac{250.000}{1}[/tex]

  Js = 3.000.000 cm : 100.000 = ... km

       = 30 km

2. Jp = S x Js

        = [tex]\frac{1}{1.000.000}[/tex] x 265

        = 0,000265 km x 100.000 = ... cm

        = 26,5 cm

3. S = 1 : [tex]\frac{Js}{Jp}[/tex]

      = 1 : [tex]\frac{320x100.000}{8}[/tex]

      = 1 : [tex]\frac{32.000.000}{8}[/tex]

      = 1 : 4.000.000

4. Js = Jp : S

        = 8 : [tex]\frac{1}{400.000}[/tex]

        = 8 x [tex]\frac{400.000}{1}[/tex]

        = 3.200.000 cm : 100.000 = ... km

1. Pengertian Bentuk Aljabar

untuk memahami bentuk aljabar ,coba kalian perhatikan permasalahan berikut.

jumlah dua bilangan bulat positif yang berurutan adalah 15.Bilangan bilangan berapakah itu?

Untuk menjawab permasalahan diatas kalian dapat membuat suatu kalimat matematika. misalkan bilangan pertama n dan bilangan berikutnya ( n + 1 ), sehingga kalimat matematika yang dimaksud adalah n ( n + 1 ) = 15

Nah ,n + ( n + 1 ) merupakan contoh bentuk aljabar. pada bentuk aljabar tersebut, terdapat nilai yang belum diketahui atau akan dicari, yaitu n . Nilai yang belum diketahui tersebut dinamakan variabel.

contoh bentuk aljabar yang lain yaitu 2x: -3p: 4y + 5: dan 2x² -3x + 7.Huruf huruf x, p dan y pada bentuk aljabar tersebut merupakan variabel. jadi, bentuk aljabar merupakan suatu model matematika yang dalam penyajiannya mengandung variabel .

seringkali kita menjumpai permasalahan yang tampak begitu rumit, akan tetapi permasalahan tersebut dapat disajikan ke modal matematika yaitu dengan bentuk aljabar.Penyajian ini sehingga permasalahan lebih mudah diselesaikan

Bentuk aljabar terdiri dari konstanta (nilainya tetap) dan variabel ( nilainya berubah ubah) melalui operasi penjumlahan, pengurangan, perkalian, pembagian,perpangkatan,dan pengakaran, Perhatikan bentuk aljabar berikut.
